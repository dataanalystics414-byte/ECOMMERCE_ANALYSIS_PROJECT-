# 🛒 E-Commerce Sales & Profit Analysis Dashboard

## 📌 Project Overview

This project presents an interactive E-Commerce Dashboard built using Power BI to analyze sales performance, profitability, customer behavior, and payment trends. The dashboard transforms raw transactional data into actionable business insights through data cleaning, modeling, and visualization techniques.

The primary goal of this project is to simulate a real-world business intelligence workflow, enabling stakeholders to monitor key performance indicators and make informed decisions.

---

## 🎯 Objectives

- Analyze overall sales, cost, and profit performance.
- Identify top-performing product categories.
- Compare profitability across different regions.
- Understand customer purchasing patterns.
- Evaluate payment method preferences.
- Build an interactive dashboard for business decision-making.

✔️ Fashion, Sports, Grocery, and Electronics contribute significantly to revenue.
